Visual Studio

I denne leksjonen skal vise hvordan vi setter opp og bruker Visual Studio 2017 i et gruppeprosjekt der vi lager en nettside.

Installasjon av Visual Studio

Hvis du ikke har Visual Studio installert allerede kan du følge denne korte videoen for å se hvordan man setter opp utviklingsmiljøet. Under følger en kort tekstversjon av videoen:
  1. Last ned Visual Studio 2017 Community herfra: https://visualstudio.microsoft.com/downloads
  2. Husk å kryss av for de tilleggsinnstillingene du ønsker når du installerer
  3. Husk også å gå på tabben "Individual Components" og å velge Github utvidelsen
  4. Start installasjonen.

Hvis du allerede har Visual Studio 2017 installert

Hvis du allerede har Visual Studio 2017 installert på maskinen din kan du følge denne guiden for å kun installere Github tilleggspakken:
  1. Trykk på "Tools"
  2. Trykk på "Extensions and Updates..."
  3. Søk etter "Github" i søkefeltet øverst til høyre
  4. Installer utvidelsen som dukker opp som resultat av søket

Koble Visual Studio 2017 med Github

Comitte endringer

Når du er ferdig med å gjøre endringer på prosjektet ditt må du comitte endringer for at andre på gruppen skal kunne se dem. Teksten og videoen under forklarer deg hvordan du går frem for å gjøre dette:
  1. Naviger deg til "Team Explorer" tabben
  2. Dobbelklikk på det Github Repositoriet du jobber med
  3. Trykk deg inn på "Changes" siden
  4. Skriv en commit melding om hva du har endret
  5. Velg filer eller trykk "Commit All"
  6. Trykk "Sync" eller gå inn på sync tabben fra "Home" siden til Github Repoet
  7. Trykk "Sync" for å synkronisere endringene dine med resten av gruppen din
