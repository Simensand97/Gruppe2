Github-opplæringspakke

Samhandling med Github

Det eneste kriteriet for å kunne bruke Github er at man har en brukerkonto som er lagt til på nettstedet. Dette gjør Github til et fint verktøy til bruk i en gruppe med mennesker som skal samhandle om å ferdigstille et produkt eller en løsning. Gjennom disse korte videosnuttene viser vi til forskjellige aspekter ved Github og hvordan vi implementerer bruken av Github for å kunne produsere og ferdigstille vår opplæringspakke.
